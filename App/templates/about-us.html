About Us




Busog-Laag Sa Cebu is a website application designed for tourist and Filipinos planning to look for places to enjoy and food delicacies to experience. The website does not only show carious restaurants but also famous tourist spots within Cebu. The website provides smart search for places and delicacies using Google map integration and allowing users to give reviews.

The program is a website application where users, local or foreign, can search places or food delicacies that is native only in Cebu province. Also the website application allows reviews for the places and foods that they have already tried. Upon visiting the website, users would then be welcome by a greeting message for Cebuâ€™s best tourist spots and delicacies. Within the homepage screen, the users would then be able to choose or search between looking for tourist spots or delicacies. Another feature added to the website is that users who don't have any access to vehicle transportation are asked and given suggestions for a cheap and comfortable ride. The main function of the website application is to search tourist spots and food delicacies in Cebu province.
